pyhealth.tasks.temporal_hospital_mortality_prediction
====================================================

Overview
--------

TemporalHospitalMortalityPredictionMIMIC3 defines an admission-level mortality
prediction task with temporal metadata, using diagnoses, procedures, medications,
and normalized admission year.
