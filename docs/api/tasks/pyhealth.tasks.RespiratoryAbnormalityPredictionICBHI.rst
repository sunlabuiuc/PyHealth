pyhealth.tasks.RespiratoryAbnormalityPredictionICBHI
=====================================================

Binary respiratory-abnormality prediction from ICBHI 2017 cycle-level audio
events. Three ``label_mode`` ablations (``"any_abnormal"``, ``"crackle_only"``,
``"wheeze_only"``) let you isolate different abnormality definitions without
re-running the data pipeline.
