pyhealth.datasets.Wav2SleepDataset
===================================

Overview
--------

A unified dataset of polysomnography (PSG) recordings spanning 7 datasets
hosted on sleepdata.org: SHHS, MESA, WSC, CHAT, CFS, CCSHS, and MROS.
Used in wav2sleep: A Unified Multi-Modal Approach to Sleep Stage
Classification from Physiological Signals
(https://arxiv.org/abs/2411.04644). Requires a Data Use Agreement via
sleepdata.org.
