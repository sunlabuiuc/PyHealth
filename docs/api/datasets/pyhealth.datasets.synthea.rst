pyhealth.datasets.synthea
=========================

Overview
--------

Synthea is an open-source synthetic patient generator that produces
realistic (but not real) patient medical records. This dataset loader
supports the standard Synthea CSV export format with conditions,
medications, procedures, and encounters tables.

Reference: Walonoski et al., "Synthea: An approach, method, and software
mechanism for generating synthetic patients and the synthetic electronic
health care record", JAMIA, 2017.

Used in: Poulain et al., "Few-Shot Learning with Semi-Supervised
Transformers for Electronic Health Records", MLHC 2022.
