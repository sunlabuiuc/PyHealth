pyhealth.datasets.NCBIDiseaseDataset
===================================

The NCBI Disease dataset provides PubMed titles and abstracts annotated with
disease mention spans and normalized concept identifiers. For more information
see `NCBI Disease Corpus <https://www.ncbi.nlm.nih.gov/CBBresearch/Dogan/DISEASE/>`_.
