pyhealth.datasets.DermoscopyDataset
===================================

The Dermoscopy dataset loader, designed to parse ISIC 2018, HAM10000, and PH2 data structures for melanoma classification. For more information, see the `isic archive <https://challenge.isic-archive.com/data/#2018>`_ and `here <https://www.fc.up.pt/addi/ph2 %20database.html>`_.
