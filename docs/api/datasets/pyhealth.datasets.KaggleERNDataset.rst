pyhealth.datasets.KaggleERNDataset
==================================

The Kaggle Inria BCI Challenge dataset (ERN/EEG). For more information see `here <https://www.kaggle.com/c/inria-bci-challenge/data>`_.

We provide an end-to-end fine-tuning example script for this dataset:
