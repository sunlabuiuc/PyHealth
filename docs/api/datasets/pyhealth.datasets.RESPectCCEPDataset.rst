pyhealth.datasets.RESPectCCEPDataset
======================================

The RESPect CCEP dataset consists of Cortico-Cortical Evoked Potentials (CCEPs) measured with Electro-CorticoGraphy (ECoG) during single pulse electrical stimulation (SPES) in 74 patients. This dataset is part of the RESPect (Registry for Epilepsy Surgery Patients) database from the University Medical Center of Utrecht.
For more information and to access the raw data, see the `OpenNeuro Repository <https://github.com/OpenNeuroDatasets/ds004080>`_.
