pyhealth.datasets.MIMIC3TLSDataset
=====================================

Dataset adapter for MIMIC-III data preprocessed by the TLS (Time-series
Learning from Scratch) pipeline. Provides feature grouping metadata for
organ-based and type-based clinical feature grouping used in step-wise
embedding models.

Reference: Kuznetsova et al., "On the Importance of Step-wise Embeddings
for Heterogeneous Clinical Time-Series", JMLR 2023.
