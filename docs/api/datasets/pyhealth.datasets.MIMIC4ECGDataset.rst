pyhealth.datasets.MIMIC4ECGDataset
===================================

The open Medical Information Mart for Intensive Care Electrocardiography (MIMIC-IV-ECG) database, refer to `doc <https://physionet.org/content/mimic-iv-ecg/1.0/>`_ for more information. We process this database into well-structured dataset object and give user the **best flexibility and convenience** for supporting modeling and analysis.
