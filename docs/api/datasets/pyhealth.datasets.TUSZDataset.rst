pyhealth.datasets.TUSZDataset
===================================

Dataset is available at https://isip.piconepress.com/projects/tuh_eeg/html/downloads.shtml

This corpus has EEG signals that have been manually annotated data for seizure events (start time, stop, channel and seizure type). For more information about this corpus, please refer to the book section (https://isip.piconepress.com/publications/book_sections/2018/frontiers_neuroscience/tuh_eeg/). The electrode configurations and annotation guidelines are described here (https://isip.piconepress.com/publications/reports/2020/tuh_eeg/).
