pyhealth.models.Wav2Sleep
==========================

Overview
--------

The complete Wav2Sleep model for sleep stage classification from
polysomnography biosignals. Consists of per-modality Signal Encoders,
an Epoch Mixer for cross-modal fusion, and a Sequence Mixer for temporal
modeling. A trained model can be applied to any subset of the modalities
seen during training.
