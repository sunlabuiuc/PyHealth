pyhealth.models.temporal_fusion_mlp
==================================

Overview
--------

TemporalFusionMLP is a lightweight multilayer perceptron for admission-level
mortality prediction. It is designed for temporal robustness experiments and
supports nonlinear interactions among structured EHR features.
